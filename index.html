About this mod


The original mod was created by a MinecraftForum user called Zaggy1024, however all development of the mod has discontinued on November 26, 2014 following a formal declaration stating that this mod will not be further developed.

Not all features and items from the original mod are going to be included. This is done to help build upon and improve the original mod, removing features will be avoided whenever possible in favour of adding them as a "creative mode only" feature. Along with this, many new features and items will be implemented. These implementations will favour realism, so if you're looking for a mod that is primarily oriented around fun and fast-paced action, this mod is not for you.

Here's some of the features the reborn version of the mod will introduce:

  • New bow and arrow types - each with their own advantages and disadvantages.
  • Support archery items - examples: quiver, shooting gloves, thumb rings, finger pads...
  • Custom bow and arrow textures - a true bowyer knows that looks are as important as speed.
  • Unique bow customization - improve both aesthetics and functionality of your bows.
  • Realistic archery mechanics - like difference in arrow accuracy and speed.
  • Plenty of humour - that's one thing this mod will not lack, I promise.

Keep in mind that most game mechanics will be modular, which means you will be able to turn them on and off in the configuration file. This gives you the freedom of mod customization.
